## Advanced Checkbox Group

You can create group with **AdvancedCheckbox** with component **CheckboxGroup**.

<Paper>
  <CheckboxGroupAdvanced />
</Paper>

```tsx
import React from "react";
import { AdvancedCheckbox, CheckboxGroup } from "rizzui";

export default function App() {
  const [values, setValues] = React.useState<string[]>(["one"]);

  return (
    <CheckboxGroup values={values} setValues={setValues} className="flex gap-4">
      <AdvancedCheckbox
        value="any"
        className="rounded-full border border-gray-300 px-6 py-1.5 hover:cursor-pointer hover:border-gray-900 peer-checked:border-gray-900 peer-checked:bg-gray-900 peer-checked:text-gray-50"
```

### AdvancedCheckbox Props

Here is the API documentation of the AdvancedCheckbox component. And the rest of the props of AdvancedCheckbox are the same as the original html input field. You can use props like **value**, **name**, **disabled** etc.

<div className="overflow-x-auto">

|      Props      |          Type           |                        Description                         | Default |
| :-------------: | :---------------------: | :--------------------------------------------------------: | :-----: |
|    className    |        `string`         |            Add className to style the container            |  \_ \_  |
| inputClassName  |        `string`         |           Add inputClassName to style the input            |  \_ \_  |
|    children     |       `ReactNode`       |                  Pass content as children                  |  \_ \_  |
|   helperText    |       `ReactNode`       |  Add helper text. It could be string or a React component  |  \_ \_  |
|      error      |        `string`         |             Show error message using this prop             |  \_ \_  |
| errorClassName  |        `string`         | This prop allows you to customize the error message style  |  \_ \_  |
| helperClassName |        `string`         | This prop allows you to customize the helper message style |  \_ \_  |
|       ref       | `Ref<HTMLInputElement>` |                                                            |  \_ \_  |
|       key       |      `key` `null`       |                                                            |  \_ \_  |

</div>
