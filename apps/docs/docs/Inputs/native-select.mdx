# NativeSelect

A basic widget for selecting value from options.

```tsx
import { NativeSelect } from "rizzui";
```

## With Clear Button

You can clear your selected options using **clearable** & **onClear** property.

<Paper className="block">
  <NativeSelectDefault label="Clearable" clearable />
</Paper>

```tsx
import { useState } from "react";
import { NativeSelect } from "rizzui";

export default function App() {
  const [state, setState] = React.useState("John");
  return (
    <NativeSelect
      label="Name"
      value={state}
      options={["John", "Miller", "James", "Franky"]}
      onChange={(e) => setState(e.target.value)}
      clearable={!!state}
      onClear={() => setState("")}
    />
  );
}
```

## With Helper Text

You can add helper text to the **NativeSelect** component using **helperText** property.

<Paper className="block">
  <NativeSelectDefault
    label="Select Name"
    helperText="Please enter your full name"
  />
</Paper>

```tsx
import { useState } from "react";
import { NativeSelect } from "rizzui";

const options = [
  {
    id: 1,
    name: "John",
    value: "John",
  },
  {
    age: 20,
    id: 2,
    name: "Doe",
    value: "Doe",
  },
  {
    class: 12,
    id: 3,
    name: "Miller",
    value: "Miller",
  },
  {
    id: 4,
    name: "James",
    value: "James",
  },
  {
    disabled: true,
    id: 5,
    name: "Franky",
    value: "Franky",
  },
];

export default function App() {
  const [value, setValue] = useState([]);

  return (
    <NativeSelect
      label="Select Name"
      options={options}
      selectedValue={value}
      setSelectedValue={setValue}
      placeholder="Choose option from below"
      helperText="Please enter your full name"
      dropDownIcon={<ChevronDownIcon className="w-6" />}
    />
  );
}
```

### NativeSelect Props

Here is the API documentation of the Native Select component. And the rest of the props of Native Select are the same as the original html select tag. You can use props like **disabled**, **name** etc.

<div className="overflow-x-auto">

|      Props      |                      Type                      |                                                                   Description                                                                    |   Default   |
| :-------------: | :--------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------: | :---------: |
|     options     |   [`(string OR OptionType)[]`](#optiontype)    |                                                                Options for select                                                                |    \_ \_    |
|      label      |                  `ReactNode`                   |                                                                 Set field label                                                                  |    \_ \_    |
|   placeholder   |                    `string`                    |                                                           Set select placeholder text                                                            |    \_ \_    |
|     variant     | [NativeSelectVariants](#nativeselect-variants) |                                                        The variants of the component are:                                                        | `"outline"` |
|      size       |    [NativeSelectSizes](#nativeselect-sizes)    |                                    The size of the component. "sm" is equivalent to the dense select styling.                                    | `"DEFAULT"` |
|     rounded     |  [NativeSelectRounded](#nativeselect-rounded)  |                                                            The rounded variants are:                                                             | `"DEFAULT"` |
|      color      |   [NativeSelectColors](#nativeselect-colors)   |                                                               Change select color                                                                | `"DEFAULT"` |
|    disabled     |                   `boolean`                    |                                                          Whether the select is disabled                                                          |    \_ \_    |
|    clearable    |                   `boolean`                    |                                                               Add clearable option                                                               |    \_ \_    |
|     onClear     |  [NativeSelectOnclear](#nativeselect-onclear)  |                                                                   clear event                                                                    |    \_ \_    |
|     prefix      |                  `ReactNode`                   | The prefix is design for adding any icon or text on the NativeSelect field's start (it's a left icon for the `ltr` and right icon for the `rtl`) |    \_ \_    |
|     suffix      |                  `ReactNode`                   |  The suffix is design for adding any icon or text on the NativeSelect field's end (it's a right icon for the `ltr` and left icon for the `rtl`)  |    \_ \_    |
|   helperText    |                  `ReactNode`                   |                                             Add helper text. It could be string or a React component                                             |    \_ \_    |
|      error      |                    `string`                    |                                                        Show error message using this prop                                                        |    \_ \_    |
| labelClassName  |                    `string`                    |                                                       Override default CSS style of label                                                        |    \_ \_    |
| selectClassName |                    `string`                    |                                                       Override default CSS style of select                                                       |    \_ \_    |
| prefixClassName |                    `string`                    |                                                       Override default CSS style of prefix                                                       |    \_ \_    |
| suffixClassName |                    `string`                    |                                                       Override default CSS style of suffix                                                       |    \_ \_    |
| helperClassName |                    `string`                    |                                                     Override default CSS style of helperText                                                     |    \_ \_    |
| errorClassName  |                    `string`                    |                                                   Override default CSS style of error message                                                    |    \_ \_    |
|       ref       |            `Ref<HTMLSelectElement>`            |                                                                                                                                                  |    \_ \_    |
