# AdvancedCheckbox

A basic widget for getting the user input of checkbox with advanced design. Here is the API documentation of the AdvancedCheckbox component. And the rest of the props of AdvancedCheckbox are the same as the original html input field. You can use props like `value`, `name`, `disabled` etc.

```jsx
import { AdvancedCheckbox } from '@redq/rizz';
```

### AdvancedCheckbox Props

Here is the API documentation of the AdvancedCheckbox component.

|      Props      |          Type           |                        Description                         | Default |
| :-------------: | :---------------------: | :--------------------------------------------------------: | :-----: |
|    className    |        `string`         |            Add className to style the container            |  \_ \_  |
| inputClassName  |        `string`         |           Add inputClassName to style the input            |  \_ \_  |
|    children     |       `ReactNode`       |                  Pass content as children                  |  \_ \_  |
|   helperText    |       `ReactNode`       |  Add helper text. It could be string or a React component  |  \_ \_  |
|      error      |        `string`         |             Show error message using this prop             |  \_ \_  |
| errorClassName  |        `string`         | This prop allows you to customize the error message style  |  \_ \_  |
| helperClassName |        `string`         | This prop allows you to customize the helper message style |  \_ \_  |
|       ref       | `Ref<HTMLInputElement>` |                                                            |  \_ \_  |
|       key       |      `key` `null`       |                                                            |  \_ \_  |
