# AdvancedRadio

A basic widget for getting the user input of radio with advanced design. Here is the API documentation of the AdvancedRadio component. And the rest of the props of AdvancedRadio are the same as the original html input field. You can use props like `value`, `name`, `disabled` etc.

```jsx
import { AdvancedRadio } from '@redq/rizz';
```

### AdvancedRadio Props

Here is the API documentation of the AdvancedRadio component.

|     Props      |          Type           |              Description              | Default |
| :------------: | :---------------------: | :-----------------------------------: | :-----: |
|   className    |        `string`         | Add className to style the container  |  \_ \_  |
| inputClassName |        `string`         | Add inputClassName to style the input |  \_ \_  |
|    children    |       `ReactNode`       |       Pass content as children        |  \_ \_  |
|      ref       | `Ref<HTMLInputElement>` |                                       |  \_ \_  |
|      key       |      `key` `null`       |                                       |  \_ \_  |
