# AdvancedSelect

AdvancedSelect is a combination of a selectbox and an auto-complete feature. AdvancedSelect allows user to select single or multiple values from options and search for them in the input field. We used `headlessUI combobox` to build the component. For more info, see their [official documentation](https://headlessui.com/react/combobox).

```jsx
import { AdvancedSelect } from "@redq/rizz";
```

### AdvancedSelect Props

Here is the API documentation of the AdvancedSelect component.

|          Props          |                        Type                        |                   Description                    |   Default   |
| :---------------------: | :------------------------------------------------: | :----------------------------------------------: | :---------: |
|         options         |         `AdvancedSelectOptionDataTypes[]`          |                Change input color                |    \_ \_    |
|          label          |                    `ReactNode`                     |              label of the component              |    \_ \_    |
|          value          |   [AdvancedSelectValue](#advanced-select-value)    |                 Pass value state                 |    \_ \_    |
|      defaultValue       |                       `{}[]`                       |                                                  |    \_ \_    |
|        nullable         |                      `false`                       |                                                  |    \_ \_    |
|        multiple         |                     `boolean`                      |                                                  |    \_ \_    |
|        onChange         |             `((value: {}[]) => void)`              |                                                  |    \_ \_    |
|           by            |      `((a: unknown, b: unknown) => boolean)`       |                                                  |    \_ \_    |
|           as            |                 `ElementType<any>`                 |                                                  |    \_ \_    |
|         refName         |                      `string`                      |                                                  |    \_ \_    |
|        className        |                      `string`                      |  Pass className to style the combobox containe   |    \_ \_    |
|        disabled         |                     `boolean`                      |                                                  |    \_ \_    |
|      \_\_demoMode       |                     `boolean`                      |                                                  |    \_ \_    |
|          name           |                      `string`                      |                                                  |    \_ \_    |
|         variant         | [AdvancedSelectVariant](#advanced-select-variants) |        The variants of the component are:        | `"outline"` |
|          size           |   [AdvancedSelectSizes](#advanced-select-sizes)    |              Size of the component               | `"DEFAULT"` |
|         rounded         | [AdvancedSelectRounded](#advanced-select-rounded)  |            The rounded variants are:             | `"DEFAULT"` |
|          color          |   [AdvancedSelectColor](#advanced-select-color)    |              Change combobox color               | `"DEFAULT"` |
|     startInputIcon      |                    `ReactNode`                     | Provide start icon to be placed at the beginning |    \_ \_    |
|      endInputIcon       |                    `ReactNode`                     |     Provide end icon to be placed at the end     |    \_ \_    |
|     transitionProps     |                `TransitionClasses`                 |    Pass transition props to animate combobox     |    \_ \_    |
|       isCreatable       |                     `boolean`                      |              To create custom value              |   `false`   |
|     labelClassName      |                      `string`                      |      Pass labelClassName to style the label      |    \_ \_    |
|      tagClassName       |                      `string`                      | Pass to style the multi selected tag at the top  |    \_ \_    |
|     inputClassName      |                      `string`                      |      Pass inputClassName to style the input      |    \_ \_    |
|   creatableClassName    |                      `string`                      |            To style creatable option             |    \_ \_    |
|    notFoundClassName    |                      `string`                      |   To style the div if search result not found    |    \_ \_    |
|     optionClassName     |                      `string`                      |    Pass optionClassName to style each option     |    \_ \_    |
| selectedOptionClassName |                      `string`                      |        To style the selected option only         |    \_ \_    |
| CheckmarkIconClassName  |                      `string`                      |    To style the check icon of selected option    |    \_ \_    |
