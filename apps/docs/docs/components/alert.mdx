
## With Clear Button

You can create a clearable **Alert** with property **clearable**.

<Paper className="text-gray-1000 justify-start items-start gap-4">
	<AlertClearable />
</Paper>

```tsx
import React from 'react';
import { Alert, Button } from '@redq/rizzui';

export default function App() {
	const [state, setState] = React.useState(true);
	return (
		<>
			<Button
				color="info"
				onClick={() => setState(true)}
				className="tracking-wider"
			>
				Info Alert
			</Button>
			{state && (
				<Alert
					color="info"
					variant="flat"
					clearable
					onClear={() => setState(false)}
				>
					<p className="font-semibold">Alert with info</p>
					<p>
						Lorem, ipsum dolor sit amet consectetur adipisicing elit. Dolores.
					</p>
				</Alert>
			)}
		</>
	);
}
```

### Alert Props

Here is the API documentation of the Alert component.

|         Props          |              Type              |                    Description                    |   Default   |
| :--------------------: | :----------------------------: | :-----------------------------------------------: | :---------: |
|         color          |   [AlertColor](#alert-color)   |                Change input color                 |    \_ \_    |
|        children        |          `ReactNode`           |          Pass alert message as children           |    \_ \_    |
|          size          |    [AlertSize](#alert-size)    |            The size of the component.             | `"DEFAULT"` |
|        rounded         | [AlertRounded](#alert-rounded) |             The rounded variants are:             | `"DEFAULT"` |
|        variant         |        `flat` `outline`        |        The variants of the component are:         | `"outline"` |
|          bar           |           `boolean`            |        Whether left bar should be visible         |  `"false"`  |
|       clearable        |           `boolean`            |               Add clearable option                |    \_ \_    |
|        onClear         | [AlertOnclear](#alert-onclear) |       Pass onClick function to clear alert        |    \_ \_    |
|       startIcon        |          `ReactNode`           | Customize start icon according to your preference |    \_ \_    |
|        endIcon         |          `ReactNode`           | Customize right icon according to your preference |    \_ \_    |
|       className        |            `string`            |       Add className to design the container       |    \_ \_    |
|      barClassName      |            `string`            |      Add barClassName to design the left bar      |    \_ \_    |
| iconContainerClassName |            `string`            | Add iconContainerClassName to position the icons  |    \_ \_    |
|     iconClassName      |            `string`            |   Add iconClassName to design the default icons   |    \_ \_    |
