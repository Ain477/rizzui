# Badge

Badge is a small overlapped UI item which indicates a status, notification, or event that appears in relativity with the underlying object.

```tsx
import { Badge } from 'rizzui';
```

<div className="mb-20 md:mb-28" />

## Default

The default style of **Badge** component.

<Paper>
	<Badge>Badge</Badge>
</Paper>

```tsx
import { Badge } from 'rizzui';

export default function App() {
	return <Badge>Badge</Badge>;
}
```

### Badge Props

Here is the API documentation of the Badge component. You can use props like **id**, **title**, **onClick** etc.

<div className="overflow-x-auto">

|       Props       |               Type               |                                      Description                                       |   Default   |
| :---------------: | :------------------------------: | :------------------------------------------------------------------------------------: | :---------: |
|     children      |        `React.ReactNode`         |                          Accepts everything React can render                           |    \_ \_    |
|      variant      | [BadgeVariants](#badge-variants) |                                 Set the Badge variants                                 |  `"solid"`  |
|       size        |    [BadgeSizes](#badge-sizes)    | Set the size of the component. <br /> `"sm"` is equivalent to the dense Badge styling. | `"DEFAULT"` |
|      rounded      |  [BadgeRounded](#badge-rounded)  |                        Set border radius of the Badge component                        | `"DEFAULT"` |
|       color       |   [BadgeColors](#badge-colors)   |                                   Change Badge color                                   | `"DEFAULT"` |
| enableOutlineRing |            `boolean`             |                Set a outline ring. It is useful for the overlapping UI.                |    \_ \_    |
|    renderAsDot    |            `boolean`             |                                 Render badge as a dot                                  |    \_ \_    |
|     className     |             `string`             |                           Add custom classes for extra style                           |    \_ \_    |

</div>
