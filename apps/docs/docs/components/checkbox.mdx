# Checkbox

A basic widget for getting the user input of checkbox. Here is the API documentation of the Checkbox component. And the rest of the props of Checkbox are the same as the original html input field. You can use props like `value`, `disabled`, `onChange`, `onFocus`, `onBlur` etc.

```tsx
import { Checkbox } from '@redq/rizzui';
```

<div className="mb-28" />

## Default

The default style of the **Checkbox**

<Paper>
	<Checkbox className="m-2" label="Yes" value="yes" />
</Paper>

```tsx
import { Checkbox } from '@redq/rizzui';

export default function App() {
	return <Checkbox className="m-2" label="Yes" value="yes" />;
}
```

## With Helper Text

You can add helper text to **Checkbox**.

<Paper>
	<Checkbox
		className="m-2"
		helperText="Hi! My name is John Doe."
		label="John"
		value="john"
	/>
</Paper>

```tsx
import { Checkbox } from '@redq/rizzui';

export default function App() {
	return (
		<Checkbox
			className="m-2"
			helperText="Hi! My name is John Doe."
			label="John"
			value="john"
		/>
	);
}
```
