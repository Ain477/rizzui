# Checkbox

A basic widget for getting the user input of checkbox.

```tsx
import { Checkbox } from 'rizzui';
```

<div className="mb-28" />

## Default

The default style of **Checkbox** component.

<Paper className="text-gray-1000">
	<Checkbox label="Remember me" />
</Paper>


## Checkbox Group

You can make a group of multiple choices **Checkbox** with component **CheckboxGroup**.

<Paper>
	<CheckboxGroupDefault />
</Paper>

```tsx
import { useState } from 'react';
import { Checkbox, CheckboxGroup } from 'rizzui';

export default function App() {
	const [values, setValues] = useState([]);
	return (
		<CheckboxGroup
			values={values}
			setValues={setValues}
			className="flex flex-row gap-4"
		>
			<Checkbox label="Apple" value="apple" />
			<Checkbox label="Pear" value="pear" />
			<Checkbox label="Orange" value="orange" disabled />
			<Checkbox label="Blueberry" value="blueberry" />
		</CheckboxGroup>
	);
}
```

### Checkbox Props

Here is the API documentation of the Checkbox component. And the rest of the props of Checkbox are the same as the original html input field. You can use props like `value`, `disabled`, `onChange`, `onFocus`, `onBlur` etc.

|       Props        |                  Type                  |                                         Description                                         |   Default   |
| :----------------: | :------------------------------------: | :-----------------------------------------------------------------------------------------: | :---------: |
|       label        |              `ReactNode`               |                                       Set field label                                       |    \_ \_    |
|       value        |                                        |                                                                                             |    \_ \_    |
|     className      |                `string`                |                   Use className prop to apply style for entire component                    |    \_ \_    |
|      variant       | [CheckboxVariants](#checkbox-variants) |                             The variants of the component are:                              | `"outline"` |
|        size        |    [CheckboxSizes](#checkbox-sizes)    |       The size of the component.<br /> "sm" is equivalent to the dense input styling.       | `"DEFAULT"` |
|       color        |   [CheckboxColors](#checkbox-colors)   |                                     Change input color                                      | `"DEFAULT"` |
|      rounded       |  [CheckboxRounded](#checkbox-rounded)  |                                  The rounded variants are:                                  | `"DEFAULT"` |
|   labelPlacement   |             `start` `end`              |                           Available directions of the label are:                            |   `"end"`   |
|      disabled      |               `boolean`                |                                Whether the input is disabled                                |    \_ \_    |
|       error        |                `string`                |                             Show error message using this prop                              |    \_ \_    |
|     helperText     |              `ReactNode`               |                  Add helper text. It could be string or a React component                   |    \_ \_    |
|  activeClassName   |                `string`                |       Use activeClassName prop to apply style on active component from checkboxGroup        |    \_ \_    |
| containerClassName |                `string`                | Use containerClassName prop to apply some additional style for label and checkbox container |    \_ \_    |
|   iconClassName    |                `string`                |          Use iconClassName prop to apply some additonal style for check mark icon           |    \_ \_    |
|   labelClassName   |                `string`                |          Use labelClassName prop to apply some addition style for the field label           |    \_ \_    |

|   errorClassName   |                `string`                |                  This prop allows you to customize the error message style                  |    \_ \_    |
|  helperClassName   |                `string`                |                 This prop allows you to customize the helper message style                  |    \_ \_    |
