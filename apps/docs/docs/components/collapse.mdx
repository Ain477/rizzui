# Collapse

A content area which can be collapsed and expanded. Completely unstyled, fully accessible UI components, designed to integrate beautifully with Tailwind CSS.

```tsx
import { Collapse } from 'rizzui';
```

<div className="mb-20 md:mb-28" />

## Simple Collapse

A simple style of **Collapse**.

<Paper className="block">
	<div>
		{defaultCollapseData.map((item) => (
			<Collapse
				key={item.title}
				className="mx-8 border-b last-of-type:border-b-0"
				panelClassName="mb-7"
				header={({ open, toggle }) => (
					<button
						type="button"
						onClick={toggle}
						className="flex w-full cursor-pointer items-center justify-between py-5 text-xl font-semibold"
					>
						{item.title}
						<ChevronDownIcon
							className={cn(
								'h-5 w-5 -rotate-90 transform transition-transform duration-300',
								open && '-rotate-0'
							)}
						/>
					</button>
				)}
			>
				{item.content}
			</Collapse>
		))}
	</div>
</Paper>

## With Custom Style

You can change the style of the **Collapse**.

<Paper className="block">
	<div className="rounded-3xl border">
		{customStyleCollapseData.map((item) => (
			<Collapse
				key={item.title}
				className="mx-8 my-2 border-b last-of-type:border-b-0"
				panelClassName="mb-7"

					<button
						type="button"

### Collapse Props

Here is the API documentation of the Collapse component.

<div className="overflow-x-auto">

|     Props      |                Type                |                         Description                          | Default |
| :------------: | :--------------------------------: | :----------------------------------------------------------: | :-----: |
|      tag       |            ` div``ul `             | Set HTML tag of the Collapse component. Either `div` Or `ul` | `"div"` |
|     header     | [CollapseHeader](#collapse-header) |               Header component of the Collapse               |  \_ \_  |
|    duration    |              `number`              |                   Set transition duration                    |  \_ \_  |
|  defaultOpen   |             `boolean`              |                     Initial active panel                     | `false` |
| panelClassName |              `string`              |  Add custom classes to the `Collapse` panel for extra style  |  \_ \_  |
|   className    |              `string`              | Add custom classes to the Collapse component for extra style |  \_ \_  |

</div>
