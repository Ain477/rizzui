<Paper className="flex-row flex-wrap gap-6">

					<ChevronDownIcon className="ml-2 w-5" />

					>
						{() => item}
					</DropdownItem>
				)
			)}
		</Dropdown>
	))}
</Paper>

### Dropdown Props

Here is the API documentation of the Dropdown component. You can use the following props to create a Dropdown.

|          Props          |                  Type                  |                                 Description                                  |   Default   |
| :---------------------: | :------------------------------------: | :--------------------------------------------------------------------------: | :---------: |
|           as            |           `ElementType<any>`           |                                                                              |    \_ \_    |
