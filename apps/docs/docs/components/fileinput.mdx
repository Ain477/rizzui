# FileInput

A basic widget for getting the attachment file.

```tsx
import { FileInput } from 'rizzui';
```

## With Clear Button

You can clear value of the **FileInput** using **clearable** & **onClear** property.

<Paper className="block">
	<FileInputClearable />
</Paper>

```tsx
import React from 'react';
import { FileInput } from 'rizzui';

export default function App() {
	const [state, setState] = React.useState<any>('');

	return (
		<FileInput
			value={state}
			onChange={(e) => setState(e.target.value)}
			clearable={!!state}
			onClear={() => {
				setState('');
			}}
		/>
	);
}
```

### FileInput Props

Here is the API documentation of the Input component. And the rest of the props are the same as the original html input field. You can use props like **value**, **disabled**, **placeholder**, **onChange**, **onFocus**, **onBlur** etc.

<div className="overflow-x-auto">

|      Props      |                   Type                   |                                 Description                                 |   Default   |
| :-------------: | :--------------------------------------: | :-------------------------------------------------------------------------: | :---------: |
|      label      |               `ReactNode`                |                               Set field label                               |    \_ \_    |
|     variant     | [FileinputVariants](#fileinput-variants) |                     The variants of the component are:                      | `"outline"` |
|      size       |    [FileinputSizes](#fileinput-sizes)    | The size of the component. `"sm"` is equivalent to the dense input styling. | `"DEFAULT"` |
|     rounded     |  [FileinputRounded](#fileinput-rounded)  |                          The rounded variants are:                          | `"DEFAULT"` |
|      color      |   [FileinputColors](#fileinput-colors)   |                             Change input color                              | `"DEFAULT"` |
|   placeholder   |                 `string`                 |                         Set input placeholder text                          |    \_ \_    |
|    disabled     |                `boolean`                 |                        Whether the input is disabled                        |    \_ \_    |
|    clearable    |                `boolean`                 |                            add clearable option                             |    \_ \_    |
|     onClear     |  [FileinputOnclear](#fileinput-onclear)  |                                 clear event                                 |    \_ \_    |
|   helperText    |               `ReactNode`                |          Add helper text. It could be string or a React component           |    \_ \_    |
|      error      |                 `string`                 |                     Show error message using this prop                      |    \_ \_    |
| labelClassName  |                 `string`                 |                     Override default CSS style of label                     |    \_ \_    |
| inputClassName  |                 `string`                 |                     Override default CSS style of input                     |    \_ \_    |
| helperClassName |                 `string`                 |                  Override default CSS style of helperText                   |    \_ \_    |
| errorClassName  |                 `string`                 |                 Override default CSS style of error message                 |    \_ \_    |
|       ref       |         `Ref<HTMLInputElement>`          |                                                                             |    \_ \_    |
|       key       |               `key` `null`               |                                                                             |    \_ \_    |

</div>
