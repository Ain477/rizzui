# Loader

A loader for displaying loading state of a page or a section.

```jsx
import { Loader } from 'rizzui';
```

<div className="mb-20 md:mb-28" />

## Default

The default style of the **Loader** component.

<Paper>
	<Loader />
</Paper>

```jsx
import { Loader } from 'rizzui';

export default function App() {
	return <Loader />;
}
```

<div className="mb-16" />

## Sizes

You can change the size of the **Loader** with property [**size**](#loader-sizes).

<Paper>
	<div className="flex items-center justify-around gap-3 w-full">
		<Loader size="sm" />
		<Loader />
		<Loader size="lg" />
		<Loader size="xl" />
	</div>
</Paper>

```jsx
import { Loader } from 'rizzui';

export default function App() {
	return (
		<>
			<Loader size="sm" />
			<Loader />
			<Loader size="lg" />
			<Loader size="xl" />
		</>
	);
}
```

## Animations

You can change the animation of the **Loader** with property [**animation**](#loader-animations).

<Paper>
	<div className="flex items-center justify-around gap-3 w-full">
		<Loader size="xl" animation="scaleUp" />
		<Loader size="xl" animation="blink" />
	</div>
</Paper>

```jsx
import { Loader } from 'rizzui';

export default function App() {
	return (
		<>
			<Loader size="xl" animation="scaleUp" />
			<Loader size="xl" animation="blink" />
		</>
	);
}
```

### Loader Props

Here is the API documentation of the Loader component.

<div className="overflow-x-auto">

|   Props   |                 Type                  |            Description             |   Default   |
| :-------: | :-----------------------------------: | :--------------------------------: | :---------: |
|    tag    |      [LoaderTags](#loader-tags)       |         HTML tag supports          |   `"div"`   |
|   size    |     [LoaderSizes](#loader-sizes)      |          Set loader size           | `"DEFAULT"` |
|   color   |    [LoaderColors](#loader-colors)     |        Change Loader color         | `"DEFAULT"` |
| animation | [LoaderAnimations](#loader-animation) |       Animation variants are       |  `"blink"`  |
| className |               `string`                | Add custom classes for extra style |    \_ \_    |
