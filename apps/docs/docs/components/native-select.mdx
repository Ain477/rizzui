# NativeSelect

A basic widget for user selection of the options.

```tsx
import { NativeSelect } from 'rizzui';
```

## With Clear Button

You can change the style of **NativeSelect** with property **clearable**.

<Paper className="block">
	<NativeSelectDefault label="Clearable" clearable />
</Paper>

```tsx
import { useState } from 'react';
import { NativeSelect } from 'rizzui';

export default function App() {
	const [state, setState] = React.useState('John');
	return (
		<NativeSelect
			label="Name"
			value={state}
			options={['John', 'Miller', 'James', 'Franky']}
			onChange={(e) => setState(e.target.value)}
			clearable={!!state}
			onClear={() => setState('')}
		/>
	);
}
```

```tsx
import { useState } from 'react';
import { NativeSelect } from 'rizzui';

const options = [
	{
		id: 1,
		name: 'John',
		value: 'John',
	},
	{
		age: 20,
		id: 2,
		name: 'Doe',
		value: 'Doe',
	},
	{
		class: 12,
		id: 3,
		name: 'Miller',
		value: 'Miller',
	},
	{
		id: 4,
		name: 'James',
		value: 'James',
	},
	{
		disabled: true,
		id: 5,
		name: 'Franky',
		value: 'Franky',
	},
];

export default function App() {
	const [value, setValue] = useState([]);

	return (
		<NativeSelect
			label="Select Name"
			options={options}
			selectedValue={value}
			setSelectedValue={setValue}
			placeholder="Choose option from below"
			helperText="Please enter your full name"
			dropDownIcon={<ChevronDownIcon className="w-6" />}
		/>
	);
}
```

### NativeSelect Props

Here is the API documentation of the Native Select component. And the rest of the props of Native Select are the same as the original html select tag. You can use props like **disabled**, **name** etc.

|        Props        |                      Type                      |                                                             Description                                                             |           Default            |
| :-----------------: | :--------------------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------: | :--------------------------: |
|       options       |                `OptionsType[]`                 |                                                         Options for select                                                          |            \_ \_             |
|        label        |                  `ReactNode`                   |                                                           Set field label                                                           |            \_ \_             |
|     placeholder     |                    `string`                    |                                                     Set input placeholder text                                                      | `"Select option from below"` |
|    selectedValue    |                `OptionsType[]`                 |                                                      Currently selected value                                                       |            \_ \_             |
|  setSelectedValue   |   `Dispatch<SetStateAction<OptionsType[]>>`    |                                                 Set newly selected value on change                                                  |            \_ \_             |
|      multiple       |                   `boolean`                    |                                                         Set multiple values                                                         |           `false`            |
|      disabled       |                   `boolean`                    |                                                   Whether the select is disabled                                                    |           `false`            |
|       variant       | [NativeSelectVariants](#nativeselect-variants) |                                                 The variants of the component are:                                                  |         `"outline"`          |
|        size         |    [NativeSelectSizes](#nativeselect-sizes)    |                          The size of the component. <br /> "sm" is equivalent to the dense select styling.                          |         `"DEFAULT"`          |
|       rounded       |  [NativeSelectRounded](#nativeselect-rounded)  |                                                      The rounded variants are:                                                      |         `"DEFAULT"`          |
|        color        |   [NativeSelectColors](#nativeselect-colors)   |                                                         Change input color                                                          |         `"DEFAULT"`          |
|      clearable      |                   `boolean`                    |                                                        add clearable option                                                         |           `false`            |
|    dropDownIcon     |                  `ReactNode`                   | The dropDownIcon is design for adding any icon on the Input field's end (it's left icon for the `ltr` and right icon for the `rtl`) |            \_ \_             |
|        error        |                    `string`                    |                                                 Show error message using this prop                                                  |            \_ \_             |
|     helperText      |                  `ReactNode`                   |                                      Add helper text. It could be string or a React component                                       |            \_ \_             |
|      className      |                    `string`                    |                                                  Add custom classes for container                                                   |            \_ \_             |
|   labelClassName    |                    `string`                    |                                Use labelClassName prop to do some addition style for the field label                                |            \_ \_             |
|   selectClassName   |                    `string`                    |                                                    Add custom classes for select                                                    |            \_ \_             |
|    iconClassName    |                    `string`                    |                                          This prop allows you to customize the icon style                                           |            \_ \_             |
|   errorClassName    |                    `string`                    |                                      This prop allows you to customize the error message style                                      |            \_ \_             |
| helperTextClassName |                    `string`                    |                                   This prop allows you to customize the helper text message style                                   |            \_ \_             |
|      autoFocus      |                   `boolean`                    |                                                                                                                                     |            \_ \_             |
|        form         |                    `string`                    |                                                                                                                                     |            \_ \_             |
|        name         |                    `string`                    |                                                                                                                                     |            \_ \_             |
|        value        |   [NativeSelectValues](#nativeselect-values)   |                                                                                                                                     |            \_ \_             |
|      onChange       |    `ChangeEventHandler<HTMLSelectElement>`     |                                                                                                                                     |            \_ \_             |
|    autoComplete     |                    `string`                    |                                                                                                                                     |            \_ \_             |
|      required       |                   `boolean`                    |                                                                                                                                     |            \_ \_             |
