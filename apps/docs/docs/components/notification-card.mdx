# NotificationCard

A NotificationCard component is used to show notification message.

```tsx
import { NotificationCard } from "rizzui";
```

## With React Hot Toast

You can do a clickable **NotificationCard** with **React Hot Toast**.

<Paper>
  <Button onClick={notification}>Click to get toast notification</Button>
  <Toaster />
</Paper>

```tsx
import toast, { Toaster } from "react-hot-toast";
import { NotificationCard, Button, cn } from "rizzui";
import { CheckCircleIcon } from "@heroicons/react/24/solid";

const notification = () =>
  toast.custom(
    (t) => (
      <NotificationCard
        title="Success"
        description="This is a success notification toast."
        media={<CheckCircleIcon className="h-6 w-6 text-green" />}
        variant="flat"
        color="success"
        onClear={() => toast.dismiss(t.id)}
        containerClassName={cn(
          "max-w-xs transition-all ease-in-out duration-300 bg-green-lighter",
          t.visible ? "" : "translate-x-[150%]"
        )}
        className="[&_h2]:!mb-0"
      />
    ),
    {
      duration: 5000,
      position: "top-right",
    }
  );

export default function App() {
  return (
    <>
      <Button onClick={notification}>Click to get toast notification</Button>
      <Toaster />
    </>
  );
}
```

### NotificationCard Props

Here is the API documentation of the NotificationCard component.

|       Props        |                         Type                         |                      Description                      |   Default   |
| :----------------: | :--------------------------------------------------: | :---------------------------------------------------: | :---------: |
|       title        |                       `string`                       |                Pass notification title                |    \_ \_    |
|    description     |                       `string`                       |         Pass notification message as children         |    \_ \_    |
| containerClassName |                     `className`                      |                To style the container                 |    \_ \_    |
|        size        |   [NotificationCardSizes](#notificationcard-sizes)   |              The size of the component.               | `"DEFAULT"` |
|      rounded       | [NotificationCardRounded](#notificationcard-rounded) |               The rounded variants are:               | `"DEFAULT"` |
|      variant       |                   `flat` `outline`                   |          The variants of the component are:           |  `"flat"`   |
|       color        |  [NotificationCardColors](#notificationcard-colors)  |            These are the color variations             | `"DEFAULT"` |
|      onClear       |                    `(() => void)`                    |      Pass onClear function to clear notification      |    \_ \_    |
|       media        |                     `ReactNode`                      |         Place icons, avatar or image at start         |    \_ \_    |
|       suffix       |                     `ReactNode`                      |   Customize right icon according to your preference   |    \_ \_    |
|     className      |                       `string`                       | Add className to design the notification message body |    \_ \_    |
|   mediaClassName   |                       `string`                       |               To customize media style                |    \_ \_    |
|  suffixClassName   |                       `string`                       |               To customize suffix style               |    \_ \_    |
|   iconClassName    |                       `string`                       |     Add iconClassName to design the default icon      |    \_ \_    |
