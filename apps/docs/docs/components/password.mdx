# Password

A basic widget for getting the user input.

```tsx
import { Password } from 'rizzui';
```

## Default

The default style of **Password** component.

<Paper className="text-gray-1000 block">
	<Password label="Password" placeholder="Enter your password" />
</Paper>


## Prefix & Visibility Icon

You can change the style of **Password** with property **prefix**, **visibilityToggleIcon**.

<Paper className="grid grid-cols-1 gap-4 text-gray-1000">
	<Password
		label="Password"
		color="warning"
		variant="active"
		placeholder="Enter your password"
		prefix={<LockClosedIcon className="h-[18px] w-[18px]" />}
	/>
	<Password
		color="warning"
		variant="active"
		label="Custom Visibility Icon"
		placeholder="Enter your password"
		visibilityToggleIcon={(visible) =>
			visible ? (
				<LockOpenIcon className="h-auto w-5" />
			) : (
				<LockClosedIcon className="h-auto w-5" />
			)
		}
	/>
</Paper>

```tsx
import { Password } from 'rizzui';
import { LockClosedIcon, LockOpenIcon } from '@heroicons/react/24/outline';

export default function App() {
	return (
		<>
			<Password
				label="Password"
				color="warning"
				variant="active"
				placeholder="Enter your password"
				prefix={<LockClosedIcon className="h-[18px] w-[18px]" />}
			/>
			<Password
				color="warning"
				variant="active"
				label="Custom Visibility Icon"
				placeholder="Enter your password"
				visibilityToggleIcon={(visible) =>
					visible ? (
						<LockOpenIcon className="h-auto w-5" />
					) : (
						<LockClosedIcon className="h-auto w-5" />
					)
				}
			/>
		</>
	);
}
```

## With Clear Button

You can create clearable **Password** with property **clearable**.

<Paper className="block text-gray-1000">
	<PasswordClearable label="Password" variant="active" />
</Paper>

```tsx
import { useState } from 'react';
import { Password } from 'rizzui';

export default function App() {
	const [state, setState] = useState('my_password');

	return (
		<>
			<Password
				label="Password"
				variant="active"
				value={state}
				clearable={true}
				color="primary"
				onClear={() => setState('')}
				onChange={(e) => setState(e.target.value)}
				placeholder="Your password"
			/>
		</>
	);
}
```

### Password Props

Here is the API documentation of the Password component. And the rest of the props are the same as the original html input field type password. You can use props like `value`, `disabled`, `placeholder`, `onChange`, `onFocus`, `onBlur` etc.

|             Props             |                 Type                 |                                                                Description                                                                |   Default   |
| :---------------------------: | :----------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------: | :---------: |
|             label             |             `ReactNode`              |                                                              Set field label                                                              |    \_ \_    |
|          placeholder          |               `string`               |                                                        Set input placeholder text                                                         |    \_ \_    |
|            variant            | [PasswordVariant](#password-variant) |                                                    The variants of the component are:                                                     | `"outline"` |
|             size              |    [PasswordSize](#password-size)    |                                The size of the component. `"sm"` is equivalent to the dense input styling.                                | `"DEFAULT"` |
|            rounded            | [PasswordRounded](#password-rounded) |                                                         The rounded variants are:                                                         | `"DEFAULT"` |
|             color             |   [PasswordColor](#password-color)   |                                                            Change input color                                                             | `"DEFAULT"` |
|           disabled            |              `boolean`               |                                                       Whether the input is disabled                                                       |    \_ \_    |
|           clearable           |              `boolean`               |                                                           Add clearable option                                                            |    \_ \_    |
|            onClear            | [PasswordOnclear](#password-onclear) |                                                                clear event                                                                |    \_ \_    |
|            prefix             |             `ReactNode`              | The prefix is design for adding any icon or text on the Input field's start (it's a left icon for the `ltr` and right icon for the `rtl`) |    \_ \_    |
|     visibilityToggleIcon      | `((visible: boolean) => ReactNode)`  |                                                It is the password visibility toggle icon.                                                 |    \_ \_    |
|          helperText           |             `ReactNode`              |                                         Add helper text. It could be string or a React component                                          |    \_ \_    |
|             error             |               `string`               |                                                    Show error message using this prop                                                     |    \_ \_    |
|        labelClassName         |               `string`               |                                                    Override default CSS style of label                                                    |    \_ \_    |
|        inputClassName         |               `string`               |                                                    Override default CSS style of input                                                    |    \_ \_    |
|        prefixClassName        |               `string`               |                                                   Override default CSS style of prefix                                                    |    \_ \_    |
| visibilityToggleIconClassName |               `string`               |                                       Override default CSS style of password show/hide toggle icon                                        |    \_ \_    |
|        helperClassName        |               `string`               |                                                 Override default CSS style of helperText                                                  |    \_ \_    |
|        errorClassName         |               `string`               |                                                Override default CSS style of error message                                                |    \_ \_    |
|              ref              |       `Ref<HTMLInputElement>`        |                                                                                                                                           |    \_ \_    |
|              key              |             `key` `null`             |                                                                                                                                           |    \_ \_    |
