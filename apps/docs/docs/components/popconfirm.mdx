# Popconfirm

Popconfirm is a simple dialog alert which can have details of the content. It can either have a confirm, cancel button or any action that can be triggered by the user's decision.

```tsx
import { Popconfirm } from "rizzui";
```

## Simple Popconfirm

A simple **Popconfirm** component.

<Paper>

        Click Me!

</Paper>

|         Props         |                      Type                      |                     Description                     |   Default   |
| :-------------------: | :--------------------------------------------: | :-------------------------------------------------: | :---------: |
|       children        |   [PopconfirmChildren](#popconfirm-children)   |        Pass children which will have tooltip        |    \_ \_    |
|         size          |      [PopconfirmSizes](#popconfirm-sizes)      |            Supported Tooltip sizes are:             | `"DEFAULT"` |
|        rounded        |    [PopconfirmRounded](#popconfirm-rounded)    |              The rounded variants are:              | `"DEFAULT"` |
|       placement       | [PopconfirmPlacements](#popconfirm-placements) |          Supported Tooltip Placements are:          |    \_ \_    |
|        shadow         |    [PopconfirmShadow](#popconfirm-shadows)     |           Supported tooltip shadows are:            | `"DEFAULT"` |
|       className       |                    `string`                    | Add custom classes for Tooltip container or content |    \_ \_    |
|        isOpen         |                   `boolean`                    |    Pass open state to open the floating element     |    \_ \_    |
|       setIsOpen       |      `Dispatch<SetStateAction<boolean>>`       |           Pass setOpen to set open value            |    \_ \_    |
|        content        |                  `ReactNode`                   |                 Content for tooltip                 |    \_ \_    |
| tooltipArrowClassName |                    `string`                    |        Add custom classes for Tooltip arrow         |    \_ \_    |
|       showArrow       |                   `boolean`                    |   Whether tooltip arrow should be shown or hidden   |    \_ \_    |
