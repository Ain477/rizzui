# Popconfirm

Popconfirm is a simple dialog alert which can have details of the content. It can either have a confirm, cancel button or any action that can be triggered by the user's decision.

```tsx
import { Popconfirm } from "@redq/rizzui";
```

## Simple Popconfirm

A simple **Popconfirm** component.

<Paper>
  <div className="h-44">
    <Popconfirm
      content={
        <div className="w-56 text-start">
          <span className="mt-2 inline-flex items-center gap-2 text-base">
            <InformationCircleIcon className="h-5 w-5" />
            <p className="font-medium">Delete the task</p>

          <p className="mt-2 text-sm">Are you sure you want to delete the task?</p>
          <div className="mt-3 mb-2 flex gap-3">
            <Button size="sm">Yes</Button>
            <Button
              size="sm"
              variant="outline"
            >
              No
            </Button>
          </div>
        </div>
      }
      placement="bottom-start"
    >
      <Button
        variant="text"
        className="focus:!ring-0"
      >
        Click Me!
      </Button>
    </Popconfirm>
  </div>
</Paper>
