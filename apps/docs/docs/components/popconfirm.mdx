# Popconfirm

Popconfirm is a simple dialog alert which can have details of the content. It can either have a confirm, cancel button or any action that can be triggered by the user's decision. We build this component on top of our **Tooltip** component, so all the style customization props are
avaiable in this compoment too except the **color** prop.

```tsx
import { Popconfirm } from 'rizzui';
```

## Actionable

You can create actionable **Popconfirm** using state.

<Paper>
	<PopconfirmActionable />
</Paper>

```tsx
import React from 'react';
import { Popconfirm, Button } from 'rizzui';
import { InformationCircleIcon } from '@heroicons/react/24/solid';

export default function App() {
	const [state, setState] = React.useState(false);
	return (
		<Popconfirm
			isOpen={state}
			setIsOpen={setState}
			content={
				<div className="w-56 text-start">
					<span className="mt-2 inline-flex items-center gap-2 text-base">
						<InformationCircleIcon className="h-5 w-5" />
						<p className="font-medium">Delete the task</p>
					</span>
					<p className="mt-2 text-sm">
						Are you sure you want to delete the task?
					</p>
					<div className="mt-3 mb-2 flex gap-3">
						<Button size="sm" onClick={() => setState(false)}>
							Yes
						</Button>
						<Button size="sm" variant="outline" onClick={() => setState(false)}>
							No
						</Button>
					</div>
				</div>
			}
			placement="bottom"
		>
			<Button variant="outline">Click Me!</Button>
		</Popconfirm>
	);
}
```

### Popconfirm Props

Here is the API documentation of the Popconfirm component.

<div className="overflow-x-auto">

|         Props         |                      Type                      |                     Description                     |   Default   |
| :-------------------: | :--------------------------------------------: | :-------------------------------------------------: | :---------: |
|       children        |   [PopconfirmChildren](#popconfirm-children)   |        Pass children which will have tooltip        |    \_ \_    |
|         size          |      [PopconfirmSizes](#popconfirm-sizes)      |            Supported Tooltip sizes are:             | `"DEFAULT"` |
|        rounded        |    [PopconfirmRounded](#popconfirm-rounded)    |              The rounded variants are:              | `"DEFAULT"` |
|       placement       | [PopconfirmPlacements](#popconfirm-placements) |          Supported Tooltip Placements are:          |    \_ \_    |
|        shadow         |    [PopconfirmShadow](#popconfirm-shadows)     |           Supported tooltip shadows are:            | `"DEFAULT"` |
|       className       |                    `string`                    | Add custom classes for Tooltip container or content |    \_ \_    |
|        isOpen         |                   `boolean`                    |    Pass open state to open the floating element     |    \_ \_    |
|       setIsOpen       |      `Dispatch<SetStateAction<boolean>>`       |           Pass setOpen to set open value            |    \_ \_    |
|        content        |                  `ReactNode`                   |                 Content for tooltip                 |    \_ \_    |
| tooltipArrowClassName |                    `string`                    |        Add custom classes for Tooltip arrow         |    \_ \_    |
|       showArrow       |                   `boolean`                    |   Whether tooltip arrow should be shown or hidden   |    \_ \_    |
