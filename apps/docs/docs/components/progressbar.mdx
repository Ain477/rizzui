# Progressbar

A basic widget for showing the work progress.

```tsx
import { Progressbar } from 'rizzui';
```

<div className="mb-28" />

## Default

The default style of **Progressbar**.

<Paper>
	<Progressbar value={75} />
</Paper>

```tsx
import { Progressbar } from 'rizzui';

export default function App() {
	return <Progressbar value={75} />;
}
```

<div className="mb-16" />

## Variants

You can change the style of **Progressbar** using **variant** property.

<Paper className="gap-6">
	<Progressbar variant="flat" value={50} />
	<Progressbar variant="solid" value={90} />
</Paper>

```tsx
import { Progressbar } from 'rizzui';

export default function App() {
	return (
		<>
			<Progressbar variant="flat" value={75} />
			<Progressbar variant="solid" value={75} />
		</>
	);
}
```

<div className="mb-16" />

## Sizes

You can change the size of **Progressbar** using [**size**](#progressbar-sizes) property.

<Paper className="gap-6">
	<Progressbar value={45} size="sm" />
	<Progressbar value={65} />
	<Progressbar value={75} size="lg" />
	<Progressbar value={85} size="xl" />
</Paper>

```tsx
import { Progressbar } from 'rizzui';

export default function App() {
	return (
		<>
			<Progressbar value={45} size="sm" />
			<Progressbar value={65} />
			<Progressbar value={75} size="lg" />
			<Progressbar value={85} size="xl" />
		</>
	);
}
```

<div className="mb-16" />

## Rounded

You can change the border radius of **Progressbar** using [**rounded**](#progressbar-rounded) property.

	<Progressbar value={65} />
	<Progressbar value={75} rounded="md" />
	<Progressbar value={85} rounded="lg" />
	<Progressbar value={95} rounded="none" />
</Paper>

```tsx
import { Progressbar } from 'rizzui';

export default function App() {

		<>
			<Progressbar value={45} rounded="sm" />
			<Progressbar value={65} />
			<Progressbar value={75} rounded="md" />
			<Progressbar value={85} rounded="lg" />
			<Progressbar value={95} rounded="none" />
		</>
	);
}
```

## With Label

You can set label inside the **Progressbar** using **label** property. It will only work when **size="xl"**.

<Paper>
	<Progressbar value={75} size="xl" label="75%" />
</Paper>

```tsx
import { Progressbar } from 'rizzui';

export default function App() {
	return <Progressbar value={75} size="xl" label="75%" />;
}
```

### Progressbar Props

Here is the API documentation of the Progressbar component.

<div className="overflow-x-auto">

|     Props      |                    Type                    |                                  Description                                   |   Default   |
| :------------: | :----------------------------------------: | :----------------------------------------------------------------------------: | :---------: |
|     value      |                  `number`                  |                            Percentage of filled bar                            |    \_ \_    |
|     label      |                  `string`                  | Pass label to show percentage inside bar. It will only work when **size="xl"** |    `""`     |
|    variant     |               `solid` `flat`               |                      The variants of the components are:                       | `"DEFAULT"` |
|      size      |   [ProgressbarSize](#progressbar-sizes)    |                           Size of the compoents are                            | `"DEFAULT"` |
|    rounded     | [ProgressbarRounded](#progressbar-rounded) |                           The rounded variants are:                            | `"DEFAULT"` |
|     color      |  [ProgressbarColor](#progressbar-colors)   |                             Pass color variations                              | `"DEFAULT"` |
|   className    |                  `string`                  |                     To style entire progressbar component                      |    \_\_     |
|  barClassName  |                  `string`                  |                         To style bar of the component                          |    \_\_     |
| labelClassName |                  `string`                  |                                 To style label                                 |    \_\_     |

</div>
