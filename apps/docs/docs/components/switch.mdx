# Switch

A basic widget for getting the user input of radio.

```tsx
import { Switch } from "@redq/rizz";
```

<div className="mb-28" />

## Variants

You can change style of **Switch** with property [**variant**](#switch-variant).

<Paper className="flex-row justify-around gap-4">
  <Switch
    variant="flat"
    label="Flat"
  />
  <Switch
    variant="outline"
    label="Outline"
  />
  <Switch
    variant="active"
    label="Active"
  />
</Paper>

```tsx
import { Switch } from "@redq/rizz";

export default function App() {
  return (
    <>
      <Switch
        variant="flat"
        label="Flat"
      />
      <Switch
        variant="outline"
        label="Outline"
      />
      <Switch
        variant="active"
        label="Active"
      />
    </>
  );
}
```

### Switch Props

Here is the API documentation of the Switch component. And the rest of the props are the same as the original html checkbox input field. You can use props like `value`, `disabled`, `checked`, `onChange`, `onFocus`, `onBlur` etc.

|      Props       |               Type               |                        Description                         |   Default   |
| :--------------: | :------------------------------: | :--------------------------------------------------------: | :---------: |
|      label       |           `ReactNode`            |                      Set field label                       |    \_ \_    |
|  labelPlacement  |          `left` `right`          |                   Change label direction                   |  `"right"`  |
|     variant      | [SwitchVariant](#switch-variant) |             The variants of the component are:             |  `"flat"`   |
|       size       |    [SwitchSize](#switch-size)    |                 The size of the component.                 | `"DEFAULT"` |
|     rounded      | [SwitchRounded](#switch-rounded) |                 The rounded variants are:                  |  `"pill"`   |
|      color       |   [SwitchColor](#switch-color)   |                    Change Switch color                     | `"DEFAULT"` |
|     disabled     |            `boolean`             |               Whether the switch is disabled               |   `false`   |
|      onIcon      |           `ReactNode`            |           Set custom icon when the switch is on            |    \_ \_    |
|     offIcon      |           `ReactNode`            |           Set custom icon when the switch is off           |    \_ \_    |
|    helperText    |           `ReactNode`            |  Add helper text. It could be string or a React component  |    \_ \_    |
|      error       |             `string`             |             Show error message using this prop             |    \_ \_    |
|    className     |             `string`             |  Add custom classes of component wrapper for extra style   |    \_ \_    |
|  labelClassName  |             `string`             |            Add custom classes for switch label             |    \_ \_    |
| switchClassName  |             `string`             |    Add custom classes of switch wrapper for extra style    |    \_ \_    |
| handlerClassName |             `string`             |    Add custom classes of switch handler for extra style    |    \_ \_    |
|  errorClassName  |             `string`             | This prop allows you to customize the error message style  |    \_ \_    |
| helperClassName  |             `string`             | This prop allows you to customize the helper message style |    \_ \_    |
|       ref        |     `Ref<HTMLInputElement>`      |                                                            |    \_ \_    |
|       key        |           `key` `null`           |                                                            |    \_ \_    |
