# Tag

Tag component is used to convey grouping information of elements. It is a standalone component which is commonly used for filtering or triggering some kind of action.

```tsx
import { Tag } from "@redq/rizzui";
```

<div className="mb-28" />

### Tag Props

Here is the API documentation of the Tag component.

|     Props     |             Type             |              Description               |   Default   |
| :-----------: | :--------------------------: | :------------------------------------: | :---------: |
|   children    |      `React.ReactNode`       |    Pass content of tag as children     |    \_ \_    |
|     size      |    [TagSizes](#tag-sizes)    |         Size of the component          | `"DEFAULT"` |
|    rounded    |  [TagRounded](#tag-rounded)  |       The rounded variants are:        | `"DEFAULT"` |
|    variant    | [TagVariants](#tag-variants) |   These are the variants we support    |  `"flat"`   |
|     color     |   [TagColors](#tag-colors)   |        Choose color variations         | `"DEFAULT"` |
|   clearable   |          `boolean`           |          Whether to clear tag          |   `false`   |
|    onClear    |  [TagOnclear](#tag-onclear)  |       Pass function to clear tag       |    \_ \_    |
|  cancelIcon   |         `ReactNode`          |        Pass custom cancel icon         |    \_ \_    |
|   className   |           `string`           | Pass className to customize tag design |    \_ \_    |
| iconClassName |           `string`           |   Pass iconClassName to design icon    |    \_ \_    |
