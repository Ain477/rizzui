# Tooltip

Tooltip displays informative text when users hover, focus, or click an element.

```jsx
import { Tooltip } from "@redq/rizzui";
```

<div className="mb-28" />

## Simple Tooltip

A simple **Tooltip** component.

<Paper>
  <Tooltip content="I'm your tooltip">
    <Button variant="outline">Hover Me!</Button>
  </Tooltip>
</Paper>

```tsx
import { Tooltip, Button } from "@redq/rizzui";

export default function App() {
  return (
    <Tooltip content="I'm your tooltip">
      <Button variant="outline">Hover Me!</Button>
    </Tooltip>
  );
}
```

## With Custom Content

You can use custom content inside **Tooltip**

<Paper>
  <Tooltip
    content={
      <div className="w-40 text-start">
        <span className="inline-flex items-center gap-2 text-base">
          <InformationCircleIcon className="w-5 h-5" />
          <p>Tooltip Title</p>
        </span>
        <p className="mt-2 text-sm">This is a tip to help you accomplish a task.</p>
      </div>
    }
    placement="bottom-start"
  >
    <Button variant="outline">Custom Content</Button>

### Tooltip Props

Here is the API documentation of the Tooltip component. You can use the following props to create a demo of tooltip.

|         Props         |                   Type                   |                      Description                      |       Default       |
| :-------------------: | :--------------------------------------: | :---------------------------------------------------: | :-----------------: |
|        content        |               `ReactNode`                |                  Content for tooltip                  | `This is a tooltip` |
|       placement       | [TooltipPlacements](#tooltip-placements) |           Supported Tooltip Placements are:           |      `bottom`       |
|       children        |   [TooltipChildren](#tooltip-children)   |         Pass children which will have tooltip         |        \_ \_        |
|        isOpen         |                `boolean`                 |     Pass open state to open the floating element      |        \_ \_        |
|       setIsOpen       |   `Dispatch<SetStateAction<boolean>>`    |            Pass setOpen to set open value             |        \_ \_        |
|         color         |     [TooltipColors](#tooltip-colors)     |                 Change Tooltip color                  |     `"DEFAULT"`     |
|         size          |      [TooltipSizes](#tooltip-sizes)      |             Supported Tooltip sizes are:              |     `"DEFAULT"`     |
|        rounded        |    [TooltipRounded](#tooltip-rounded)    |               The rounded variants are:               |     `"DEFAULT"`     |
|        shadow         |    [TooltipShadows](#tooltip-shadows)    |            Supported tooltip shadows are:             |     `"DEFAULT"`     |
|       className       |                 `string`                 |  Add custom classes for Tooltip container or content  |        \_ \_        |
| tooltipArrowClassName |                 `string`                 |         Add custom classes for Tooltip arrow          |        \_ \_        |
|       showArrow       |                `boolean`                 |    Whether tooltip arrow should be shown or hidden    |       `true`        |
|     isPopconfirm      |                `boolean`                 | Whether the tooltip is used as a pop-confirm componen |       `false`       |
