# Popover

Popover is a simple dialog alert which can have details of the content. It can either have a confirm, cancel button or any action that can be triggered by the user's decision.

```tsx
import { Popover } from "rizzui";
```

<div className="mb-20 md:mb-28" />

## Default

A simple **Popover** component.

<Paper>
  <Popover>
    <Popover.Trigger>
      <Button variant="outline">Open Popover</Button>
    </Popover.Trigger>
    <Popover.Content>
      <Title as="h6" className="!mb-0 font-semibold">
        This is a default popover
      </Title>
      <Text>It is opened when button is clicked. </Text>
    </Popover.Content>
  </Popover>
</Paper>

```tsx
import { Popover, Title, Text, Button } from "rizzui";

export default function App() {
  return (
    <Popover>
      <Popover.Trigger>
        <Button variant="outline">Open Popover</Button>
      </Popover.Trigger>
      <Popover.Content>
        <Title as="h6">This is a default popover</Title>
        <Text>It is opened when button is clicked. </Text>
      </Popover.Content>
    </Popover>
  );
}
```

## Actionable

You can create actionable **Popover** using state.

<Paper>
  <Popover>
    <Popover.Trigger>
      <ActionIcon variant="outline" color="danger">
        <TrashIcon className="w-5 h-auto" />
      </ActionIcon>
    </Popover.Trigger>
    <Popover.Content>
      {({ setOpen }) => (
        <div className="w-56">
          <Text as="b" className="font-medium mb-1">
            Delete the task
          </Text>
          <Text className="mb-1 text-sm">
            Are you sure you want to delete the task?
          </Text>
          <div className="flex justify-end gap-3 mb-1">
            <Button size="sm" variant="outline">
              No
            </Button>
            <Button size="sm" onClick={() => setOpen(false)}>
              Yes
            </Button>
          </div>
        </div>
      )}
    </Popover.Content>
  </Popover>
</Paper>

```tsx
import React from "react";
import { Popover, Title, Text, Button } from "rizzui";
import { TrashIcon } from "@heroicons/react/24/solid";

export default function App() {
  return (
    <Popover>
      <Popover.Trigger>
        <Button variant="outline" color="danger">
          <TrashIcon className="w-5 h-auto" />
        </Button>
      </Popover.Trigger>
      <Popover.Content>
        {({ setOpen }) => (
          <div className="w-56">
            <Title as="h6">Delete the task</Title>
            <Text>Are you sure you want to delete the task?</Text>
            <div className="flex justify-end gap-3 mb-1">
              <Button size="sm" variant="outline">
                No
              </Button>
              <Button size="sm" onClick={() => setOpen(false)}>
                Yes
              </Button>
            </div>
          </div>
        )}
      </Popover.Content>
    </Popover>
  );
}
```

#### Popover Props

Here is the API documentation of the Popover component.

<div className="overflow-x-auto">

|         Props         |                   Type                   |                     Description                     |   Default   |
| :-------------------: | :--------------------------------------: | :-------------------------------------------------: | :---------: |
|       children        |   [PopoverChildren](#popover-children)   |        Pass children which will have tooltip        |    \_ \_    |
|         size          |      [PopoverSizes](#popover-sizes)      |            Supported Tooltip sizes are:             | `"DEFAULT"` |
|        rounded        |    [PopoverRounded](#popover-rounded)    |              The rounded variants are:              | `"DEFAULT"` |
|       placement       | [PopoverPlacements](#popover-placements) |          Supported Tooltip Placements are:          |    \_ \_    |
|        shadow         |    [PopoverShadow](#popover-shadows)     |           Supported tooltip shadows are:            | `"DEFAULT"` |
|       className       |                 `string`                 | Add custom classes for Tooltip container or content |    \_ \_    |
|        isOpen         |                `boolean`                 |    Pass open state to open the floating element     |    \_ \_    |
|       setIsOpen       |   `Dispatch<SetStateAction<boolean>>`    |           Pass setOpen to set open value            |    \_ \_    |
|        content        |               `ReactNode`                |                 Content for tooltip                 |    \_ \_    |
| tooltipArrowClassName |                 `string`                 |        Add custom classes for Tooltip arrow         |    \_ \_    |
|       showArrow       |                `boolean`                 |   Whether tooltip arrow should be shown or hidden   |    \_ \_    |

</div>
